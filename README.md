# HR Analytics Project

## Project Overview
Analysed employee data to identify workforce patterns, employee attrition factors, salary trends, and department insights using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning
- Employee attrition analysis
- Department analysis
- Salary analysis
- Job role analysis
- Age distribution analysis
- Overtime and attrition analysis

## Key Insights
- Analysed employee turnover patterns.
- Identified department and job role distributions.
- Compared salary trends across departments.
- Studied overtime impact on employee retention.

IBM HR Analytics Employee Attrition Dataset
